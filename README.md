## Basic Crud menggunakan MySQL dan Express JS

## Installation

1. Clone the repository:

    ```bash
    https://github.com/Cramouchegit/Basic-Crud-ExpressJS.git
    ```

2. Navigate to the project directory:

    ```bash
    cd Basic-Crud-ExpressJS
    ```

3. Install Dependencies:

    ```bash
    npm install
    ```
4. Jalankan Project:

    ```bash
    npm run dev
    ```
